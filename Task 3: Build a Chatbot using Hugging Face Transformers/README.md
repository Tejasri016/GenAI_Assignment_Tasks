# 🤖 NLP Chatbot using Transformers

## 📌 Description
This project is a simple conversational chatbot built using Hugging Face Transformers and DialoGPT. The chatbot can interact with users in natural language and generate responses dynamically.

---

## 🚀 Features
- Interactive chatbot using console input
- Transformer-based response generation
- Continuous conversation flow
- Exit option (type "exit" or "quit")
- Hybrid approach for improved accuracy

---

## 🛠️ Technologies Used
- Python
- Hugging Face Transformers
- PyTorch
- Google Colab

---

## ⚙️ How it Works
User Input → Tokenization → Transformer Model → Response Generation → Output

---

## 🧠 Model Used
DialoGPT (pre-trained conversational model by Microsoft)

---

## 📸 Sample Output
Chatbot: Hello! I am your AI assistant. How can I help you today?  
You: Hello  
Chatbot: Hello! Nice to meet you. How can I assist you today?
